Build A Hexagonal Color Picker with CSS & Vanilla JavaScript

Lightweight Color Picker Component. Full Code Implementation.

Charmaine Chui
4 min read
·
Apr 9, 2022

In most corporate work places today, Microsoft Office products such as Microsoft PowerPoint or Microsoft Word are widely considered to be mandatory productivity tools. Hence, anyone who has dealt with them would be familiar with the following color picker interface:

On the other hand, many web developers would be aware of the default HTML color picker component rendered with: <input type='color' />

Hence, in order for Microsoft Office product users to handle a color picker they are accustomed to, I searched the web for attempts to re-enact the hexagonal interface and managed to find a few implementations which either:

1) Requires external plugins

  • jQuery Hex Colorpicker (requires jQuery)
  • hexbin.js (requires d3 — Source: d3-js-hexbin-hexagonal-color-picker

2) Is Image-Dependent and Code Heavy

  • W3Schools’ Implementation at HTML Color Picker (requires colormap image and selected hexagon image)

Both of which are functional but not suited as lightweight implementations due to heavy overhead costs. As such, this motivated me to create my very own hexagonal color picker using the unicode: ⬢ (denoted as U+2B22 and named Black Hexagon).

Implementation Steps

In order to replicate the hexagonal diagram, it is first essential to identify the no. of hexagons per line as illustrated below:

Step 1) Render the first 6 lines of hexagons

var n=0;
var hexNums=[7, 8, 9, 10, 11, 12];
var hexArrs=[];
var hexArr;
for(var hexNum of hexNums) {
  hexArr=[];
  for(n=0;n<hexNum;n++) {
      hexArr.push('⬢');
  }
  hexArrs.push(hexArr);
}

Step 2) Render the middle-most line with 13 hexagons

Code snippet is as follows (following up from the above):

hexArr=[];
for(n=0;n<13;n++) {
  hexArr.push('⬢');
}
hexArrs.push(hexArr);

Step 3) Render final 6 lines

var hexNums2=JSON.parse(JSON.stringify(hexNums));
hexNums2.reverse();
for(hexNum of hexNums2) {

  }
  hexArrs.push(hexArr);
}

To output all hexagon symbols, the following JavaScript code snippet is included at the end (where variable output is <div id='output'></div>:

var output=document.getElementById("output");
for(var hexArrLine of hexArrs) {
    var hexLine='<span class="colorHex">'+hexArrLine.join('</span><span class="colorHex">') + '</span>';
    output.insertAdjacentHTML('beforeend', '<span class="condensed-line">'+hexLine+'</span>');
}

Step 4) Styling the hexagons with CSS

In order to remove the gaps between each hexagon symbol, for example, the first line is denoted in HTML as:

<span class="condensed-line">
  <span class="colorHex">⬢</span>
  <span class="colorHex">⬢</span>
  <span class="colorHex">⬢</span>
  <span class="colorHex">⬢</span>
  <span class="colorHex">⬢</span>
  <span class="colorHex">⬢</span>
  <span class="colorHex">⬢</span>
</span>

The following CSS code is included:

span.condensed-line {
  text-align:center;
  display:block;
  font-size:20px;
  line-height: 14.65px;
  font-stretch: ultra-condensed;
}
span.colorHex {
  cursor:pointer;
}

Step 5) Assigning each hexagon by its respective colour

Finally, to render an exact replica of the colours, each hexagon is coloured accordingly via JavaScript:

Step 6) Assign data-hex attribute of each hexagon to be its hex colour value

For each <span class="colorHex">⬢</span> element, it has been tagged to its respective hex colours. Subsequently, upon selection of any hexagon value, 2 events should be triggered —

Event 1: The hex colour tagged to the hexagon is displayed

Event 2: The <span class="colorHex">⬢</span> selected should be highlighted

  • The CSS class color-selected can then be added to the specific hexagon upon its selection.
span.colorHex.color-selected {
    -webkit-text-stroke: 1.15px black !important;
}
